Export the .git file of this area to the UserDirectory/nwnx folder, or to the location of alias.

Parameters
areaThe NwArea to check.
fileNameThe filename, 16 characters or less and should be lowercase. If left blank the resref of oArea will be used.
newNameOptional new name of the area. Leave blank to use the current name.
newTagOptional new tag of the area. Leave blank to use the current tag.
aliasThe alias of the resource directory to add the .git file to. Default: 'UserDirectory/nwnx'.

<return>true if exported successfully, false if not.</return>

Export the .git file of this area to the UserDirectory/nwnx folder, or to the location of alias.

Parameters
areaThe NwArea to check.
fileNameThe filename, 16 characters or less and should be lowercase. If left blank the resref of oArea will be used.
exportVariablesIf true, local variables set on this area will be exported.
exportUUIDIf true, the UUID of this area will be exported.
objectTypeFilterFilter object types. These objects will not be exported.
aliasThe alias of the resource directory to add the .git file to. Default: 'UserDirectory/nwnx'.
objectsToIgnoreThe NwObjects we want to not export. Players are already ignored.

<return>true if exported successfully, false if not.</return>
